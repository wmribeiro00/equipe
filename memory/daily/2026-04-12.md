- Deploy na Hostinger é **manual** (sem CI/CD) — vale criar GitHub Action para automatizar futuramente.
- `docker-compose` (legado) não está instalado — usar `docker compose` (plugin moderno).

- Stark consegue diagnosticar bugs com precisão mesmo sem acesso ao código (via browser DevTools + logs de API).
- Sub-agentes funcionam muito bem para tarefas estruturadas com briefing claro.
- A operação precisa de política explícita de modelos para evitar desperdício.
